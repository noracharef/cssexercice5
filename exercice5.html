MODÈLES DE TEXTES
Pour toutes les occasions



A la recherche du modèle parfait de texte pour une occasion particulière ? Vous ne savez pas quoi écrire sur un livre d'or de mariage ? Vous ne savez pas quoi noter sur une carte d'anniversaire ? Vous ne savez pas comment formuler un message de remerciements? Vous ne savez pas quel SMS envoyer pour être original à l'occasion de la nouvelle année 2020 ? Vous cherchez une idée de texte pour une lettre ? Ce site est une mine d'inspiration pour tous ceux qui doivent écrire un message ou un texte pour une occasion particulière. Plusieurs catégories sont référencées sur le site : Noël, anniversaire, mariages, félicitations, condoléances, remerciements, etc. De plus, les sous-catégories vous permettront de peaufiner les styles d'écriture : humour, original ou plus formel. Vous pourrez vous en servir aisément pour écrire vos cartes, vos textos ou encore vos mails ! Plusieurs ressources sont également disponibles comme les citations d'auteurs célèbres, les conseils d'écriture, etc.
